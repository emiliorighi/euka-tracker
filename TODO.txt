    READS: select up to 5 bioprojects per species containing reads
        base_count
        read_count
        average_read_length = base_count / read_count
            Illumina short reads → ~100–300 bp
            Nanopore reads → 1,000–100,000+ bp
            PacBio HiFi → ~10–25 kb
        coverage_run = base_count / genome_size_species (0.1× ≤ coverage ≤ 200×)
            <0.1× → useless / incomplete
            1–50× → normal usable sequencing
            200× → over-sequencing noise
